# Educador.ai

Educador.ai is an AI-powered platform that transforms your textbook PDFs into interactive learning materials. Generate summaries, Q&A sessions, and structured notes effortlessly.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Features

-   **PDF Analysis**: Upload your textbook PDFs and let our AI analyze the content.
-   **Automated Summaries**: Get concise summaries of long chapters or entire books.
-   **Q&A Generation**: Create questions and answers based on the textbook content for effective revision.
-   **Note Creation**: Automatically generate structured notes from the PDF content.

## Tech Stack

-   [Next.js](https://nextjs.org/) - React framework for production.
-   [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript for robust applications.
-   [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
-   [Clerk](https://clerk.com/) - User management and authentication.
